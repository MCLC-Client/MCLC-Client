# Overview

MCLC is more than just a Minecraft launcher. It's a high-performance, modern platform designed for players who want a premium experience without the complexity of traditional modding tools.

High Performance

Optimized Java arguments and RAM management for maximum FPS.

Modern UI

A sleek, glass-morphism interface that feels alive.

Mod Hub

Integrated Modrinth and CurseForge support for easy installation.

# Installation

To use MCLC Launcher, you need to download the installer for Windows. The launcher is designed to be lightweight and fast, providing a premium experience for Minecraft players.

It doesn't matter which mod/plugin loader you are using. All of them are cross compatible*.

Important

Don't contact official Minecraft support for help with MCLC! Use our Discord instead.

  • Fabric
  • NeoForge
  • Forge
  • Spigot
  • Paper/Velocity
  • Quilt

Fabric

For the Fabric version of the mod, you need to have the Fabric Loader installed on your client/server.

Then copy the MCLC Core mod jar to the mods folder. The mods folder is located in /mods.

# Accounts

MCLC requires a valid Minecraft account. You can add both Microsoft (Premium) and Offline accounts in the account manager.

Adding a Microsoft Account

  1. Click on the profile icon in the top right corner.
  2. Select "Add Account".
  3. Choose Microsoft Login.
  4. Follow the instructions in the browser window to authorize MCLC.

# Instances

Instances are separate Minecraft installations. This allows you to have multiple versions or modpacks installed at the same time.

Creating a new Instance

To create a new instance, click the "+" button in the sidebar.

Vanilla

Pure Minecraft experience without modifications.

Modded

Support for Fabric, Forge, NeoForge and Quilt.

Managing Mods

You can easily manage your mods directly within the launcher.

  • 1

    Right-click on an instance and select "Manage Mods".

  • 2

    Use the built-in Modrinth browser to find and install new mods.

# Settings

Configure MCLC to fit your needs and optimize performance for your hardware.

Java & Memory

Allocate more RAM to Minecraft for better performance, especially with large modpacks.

Resolution

Set the default window size or enable full-screen mode by default.

# Troubleshooting

Common Issues

Game won't start

Ensure you have the correct Java version installed (Java 17 for 1.18+, Java 8 for 1.12.2 and below).

Authentication Error

Try removing and re-adding your Microsoft account in the account manager.

# FAQ

Is MCLC free?

Yes, MCLC is completely free to use. We believe in providing a premium experience for everyone.

Can I use Offline accounts?

Yes, you can add Offline accounts in the settings, although Microsoft accounts are recommended for skin support and multiplayer.
